# NexaFlow Getting Started Guide

## What is NexaFlow?
NexaFlow is a cloud-based project management and API integration platform that helps teams automate workflows, manage projects, and connect third-party services through a unified API gateway.

## Account Setup

### Step 1: Create Your Account
1. Visit https://app.nexaflow.io/signup
2. Enter your work email and create a password (minimum 12 characters)
3. Verify your email address using the link sent to your inbox
4. Complete your profile with your name, company, and timezone

### Step 2: Choose a Plan
- **Free**: Up to 3 projects, 1,000 API calls/month, 2 team members
- **Starter** ($29/month): Up to 20 projects, 50,000 API calls/month, 10 team members
- **Pro** ($99/month): Unlimited projects, 500,000 API calls/month, unlimited team members
- **Enterprise**: Custom pricing, dedicated support, SLA guarantee

### Step 3: Set Up Your First Project
1. Click "New Project" from the dashboard
2. Enter a project name and description
3. Select your team members
4. Choose a project template (Kanban, Scrum, or Custom)
5. Click "Create Project"

## First-Time Login Issues
If you cannot log in after signup:
- Check your spam/junk folder for the verification email
- Ensure you are using the exact email used during registration
- Try password reset via the "Forgot Password" link
- Clear browser cache and cookies, then retry
- If issues persist, contact support@nexaflow.io with your registration email

## Dashboard Overview
- **Projects Panel**: All active and archived projects
- **API Gateway**: Manage API keys and monitor usage
- **Team**: Invite and manage team members
- **Billing**: Subscription, invoices, and payment methods
- **Settings**: Account preferences, notifications, and security

## Minimum System Requirements
- Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 91+
- Internet connection: 5 Mbps or faster recommended
- Screen resolution: 1280x720 minimum
